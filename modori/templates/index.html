modori

A Korean Natural Language Processing Web API


Request with JSON

Request with the Data in JSON format.

[


  {

    "text": "아빠가 방에 들어가신다"
  

  }


]

            
[
  {
    "text": "아빠가 방에 들어가신다",
    "result":[
    ["아빠","NNG"],
    ["가","JKS"],
    ["방","NNG"],
    ["에","JKB"],
    ["들어가","VV"],
    ["신다","EP+EC"]
    ]
  }
]

And Get the processed Results Right Away

Tada! you get an accurate result of analyzed data within a few milliseconds.

Designed To Excel
